# Profile Site

A clean, modern portfolio site with sections for about, skills, projects, and contact.

## Customize
- Update the text in `index.html` with your name, bio, and project links.
- Replace the placeholder content in the projects section.
- Swap the colors in `styles.css` to match your brand.

## Run locally
Open `index.html` in your browser.

## Assets
`assets/placeholder.txt` is a placeholder. Replace it with your images or icons.
